# Quiz Program Using `if-else` in C 

A simple **quiz application written in C** that uses `if-else` statements to evaluate user answers and calculate a score.  
This project is designed for beginners to practice **conditional logic** in C programming.

## About

This program:
- Asks multiple questions to the user
- Gets user input
- Checks answers using `if-else` blocks
- Calculates a basic score and displays results

It's a great way to understand how decisions are handled in C. :contentReference[oaicite:1]{index=1}

--

Concepts Covered

- `printf()` and `scanf()` for input/output  
- `if`, `else`, and conditional logic  
- Variables and basic decision making  
- Simple scoring logic
